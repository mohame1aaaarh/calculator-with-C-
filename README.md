## مقدمة

في المشروع ده هعمل آلة حاسبة بسيطة باستخدام لغة C++ مع مكتبة `graphics.h`، اللي بتخلينا نرسم أشكال ونتحكم في الألوان والنصوص داخل نافذة جرافيك. الكود بيعتمد على رسم الأزرار والشاشة يدوياً، وبيتعامل مع ضغطات الماوس عشان نعرف المستخدم ضغط على إيه.

---

## هيكل الكود

الكود مقسم لأربع أجزاء رئيسية:

**أولاً:** المتغيرات والحالة  
**تانياً:** منطق العمليات الحسابية  
**تالتاً:** رسم الواجهة  
**رابعاً:** الحلقة الرئيسية ومعالجة الماوس

---

## أولاً: المتغيرات والحالة

```cpp
std::string display = "0";
double num1 = 0;
char   op   = 0;
bool   fresh = true;
```

دي المتغيرات اللي بتحفظ حالة الآلة الحاسبة في أي وقت:

- **`display`** → النص اللي بيتعرض على الشاشة، بيبدأ بـ `"0"`
- **`num1`** → بيحفظ أول رقم لما المستخدم يضغط على عملية زي `+` أو `-`
- **`op`** → بيحفظ العملية اللي المستخدم اختارها (`+` أو `-` أو `*` أو `/`)
- **`fresh`** → علامة بتقول هل المفروض نبدأ رقم جديد ولا نكمل على الرقم الحالي

---

## مصفوفة الأزرار

```cpp
const char* btns[4][4] = {
    {"7","8","9","/"},
    {"4","5","6","*"},
    {"1","2","3","-"},
    {"C","0","=","+"}
};
```

ده جدول 4×4 فيه كل الأزرار، مرتبة بنفس الترتيب اللي هتتعرض بيه على الشاشة. استخدمت مصفوفة ثنائية الأبعاد عشان يسهّل حساب مكان كل زرار على الشاشة باستخدام الصف والعمود.

## تانياً: منطق العمليات — دالة `press()`

```cpp
void press(const char* b) { ... }
```

الدالة دي هي قلب الآلة الحاسبة، بتاخد الزرار اللي المستخدم ضغط عليه وبتعمل الحاجة المناسبة.

### حالة زرار C (مسح)

```cpp
if (b[0] == 'C') {
    display = "0"; num1 = 0; op = 0; fresh = true;
}
```

لما المستخدم يضغط `C` بترجّع كل حاجة لأول حالتها، الشاشة بتبقى `"0"` والعملية والأرقام بترجع صفر.

### حالة العمليات الحسابية

```cpp
else if (b[0]=='+'||b[0]=='-'||b[0]=='*'||b[0]=='/') {
    num1 = std::stod(display);
    op   = b[0];
    fresh = true;
}
```

لما المستخدم يضغط عملية حسابية، بحفظ الرقم اللي على الشاشة في `num1`، وبحفظ العملية في `op`، وبحط `fresh = true` عشان المرة الجاية المستخدم يدخل رقم جديد يبدأ من الأول.

### حالة زرار = (النتيجة)

```cpp
else if (b[0] == '=') {
    double num2 = std::stod(display), res = 0;
    if (op=='+') res = num1 + num2;
    if (op=='-') res = num1 - num2;
    if (op=='*') res = num1 * num2;
    if (op=='/') res = num2 ? num1/num2 : 0;
    char buf[32];
    snprintf(buf, sizeof(buf), "%.8g", res);
    display = buf;
    op = 0; fresh = true;
}
```

هنا بياخد الرقم التاني من الشاشة، وبيعمل العملية المحفوظة، وفي حالة القسمة بيتحقق إن المقسوم عليه مش صفر عشان نتجنب الخطأ. النتيجة بتتحول لـ string وبتتعرض على الشاشة.

### حالة الأرقام

```cpp
else {
    if (fresh) { display = b; fresh = false; }
    else if (display == "0") display = b;
    else display += b;
}
```

لو `fresh` شغال، بنبدأ رقم جديد. لو الشاشة فيها `"0"` بس، بنبدلها بالرقم الجديد. غير كده بنضيف الرقم للرقم الموجود.

## تالتاً: رسم الواجهة — دالة `drawCalc()`

### مسح الشاشة

```cpp
cleardevice();
```

في كل مرة بنرسم من جديد، بنمسح كل حاجة على الشاشة الأول عشان نبدأ رسمة نظيفة.

### رسم شاشة العرض

```cpp
setfillstyle(SOLID_FILL, BLACK);
bar(10, 10, 310, 70);
setcolor(WHITE);
settextstyle(DEFAULT_FONT, HORIZ_DIR, 3);
int tw = textwidth((char*)display.c_str());
outtextxy(300 - tw, 30, (char*)display.c_str());
```

- `bar()` → بترسم مستطيل أسود اللون هو شاشة العرض
- `textwidth()` → بتحسب عرض النص عشان نعرفه على اليمين
- `outtextxy()` → بتكتب النص في مكان معين على الشاشة

### رسم الأزرار

```cpp
for (int r = 0; r < 4; r++) {
    for (int c = 0; c < 4; c++) {
        int x = 10 + c * 76;
        int y = 80 + r * 76;
        ...
    }
}
```

بستخدم loop مزدوجة على الصفوف والأعمدة. مكان كل زرار بيتحسب من الصف والعمود، كل زرار عرضه 68 وفيه مسافة 8 بين كل زرار والتاني (76 = 68 + 8).

**ألوان الأزرار:**

```cpp
if (lbl[0]=='+'||...) setfillstyle(SOLID_FILL, YELLOW);  // عمليات = أصفر
else if (lbl[0]=='=') setfillstyle(SOLID_FILL, GREEN);   // يساوي = أخضر
else if (lbl[0]=='C') setfillstyle(SOLID_FILL, RED);     // مسح = أحمر
else                   setfillstyle(SOLID_FILL, DARKGRAY); // أرقام = رمادي
```

كل نوع زرار ليه لون مختلف عشان يسهّل على المستخدم يميّز بينهم.

**توسيط النص على الزرار:**

```cpp
int lw = textwidth((char*)lbl);
int lh = textheight((char*)lbl);
outtextxy(x+(68-lw)/2, y+(68-lh)/2, (char*)lbl);
```

بحسب عرض وارتفاع النص، وبطرحهم من عرض الزرار وبقسم على 2 عشان النص يبقى في النص بالظبط.

### تحديث الشاشة

```cpp
refresh();
```

دي الدالة المهمة جداً في مكتبة SDL_bgi، من غيرها الرسمة مش بتتعرض. بتبعث كل اللي رسمناه للشاشة فعلياً.

## رابعاً: الحلقة الرئيسية

```cpp
bool mouseWasDown = false;

while (true) {
    bool mouseNowDown = ismouseclick(WM_LBUTTONDOWN);

    if (mouseNowDown && !mouseWasDown) {
        int mx = mousex();
        int my = mousey();
        clearmouseclick(WM_LBUTTONDOWN);
        // فحص كل زرار
    }

    mouseWasDown = mouseNowDown;
    delay(50);
}
```

### فكرة الـ Debounce

المشكلة اللي كانت موجودة إن الحلقة بتتكرر كل 50ms، وكل مرة لو إيد المستخدم لسه على الزرار بتحسب ضغطة جديدة. الحل إني بحفظ حالة الماوس في المرة السابقة (`mouseWasDown`) وبشغّل الكود بس لما الماوس ينتقل من **مش مضغوط → مضغوط**، يعني لما `mouseNowDown = true` و `mouseWasDown = false` في نفس الوقت.

### فحص مكان الضغطة

```cpp
for (int r = 0; r < 4; r++)
    for (int c = 0; c < 4; c++) {
        int x = 10 + c * 76;
        int y = 80 + r * 76;
        if (mx>=x && mx<=x+68 && my>=y && my<=y+68) {
            press(btns[r][c]);
            drawCalc();
        }
    }
```

بفحص كل زرار، لو مكان الضغطة جوّا حدود الزرار، بنادي `press()` بالزرار ده وبعدين بنرسم الشاشة من جديد.

### الخروج من البرنامج

```cpp
if (kbhit()) {
    char k = getch();
    if (k == 27) break;
}
```

لو المستخدم ضغط `ESC` (كود 27) الحلقة بتوقف والبرنامج بيقفل.

## ملخص دوال مكتبة graphics.h المستخدمة

| الدالة | الوظيفة |
|---|---|
| `initwindow(w, h, title)` | بتفتح نافذة الرسم بعرض وارتفاع وعنوان |
| `cleardevice()` | بتمسح كل حاجة على الشاشة |
| `setfillstyle(style, color)` | بتحدد نوع ولون التعبئة |
| `bar(x1, y1, x2, y2)` | بترسم مستطيل ملوّن |
| `setcolor(color)` | بتحدد لون الخطوط والنصوص |
| `rectangle(x1, y1, x2, y2)` | بترسم إطار مستطيل |
| `settextstyle(font, dir, size)` | بتحدد شكل وحجم الخط |
| `textwidth(str)` | بترجع عرض النص بالبكسل |
| `textheight(str)` | بترجع ارتفاع النص بالبكسل |
| `outtextxy(x, y, str)` | بتكتب نص في مكان معين |
| `refresh()` | بتعرض كل الرسمة على الشاشة |
| `ismouseclick(type)` | بتتحقق لو في ضغطة ماوس |
| `mousex() / mousey()` | بترجع مكان المؤشر |
| `clearmouseclick(type)` | بتمسح حدث الضغطة |
| `kbhit()` | بتتحقق لو في مفتاح اتضغط |
| `getch()` | بتاخد المفتاح اللي اتضغط |
| `delay(ms)` | بتوقف البرنامج لعدد من الميلي ثانية |
| `closegraph()` | بتقفل نافذة الرسم |
